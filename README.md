 **Number Guessing Game **

## 概要 
本アプリケーションは、1〜100 の範囲で生成された乱数をユーザーが推測し、正解するまで繰り返し入力を行うコンソール型の数当てゲームです。  
---
## 開発の動機 
Java 学習初期段階において、文法を断片的に覚えるのではなく、制御フロー・入力処理・状態管理を一つの処理としてまとめて理解することを目的として、本アプリを作成しました。  
また、日本企業での就業を意識し、「なぜこの構造を選択したのか」を説明できるコードを書くことを意識しています。
---
## 実装内容 / 实现内容
- 「Random」クラスを用いた 1〜100 の乱数生成  
- 「Scanner」によるユーザー入力処理  
- 「while」ループと「break」による処理継続・終了制御  
- 試行回数を管理するカウンタ変数の実装  
- 入力値と正解値の大小比較によるヒント表示  
---
## 工夫した点 
- 正解するまで処理を継続するため、無限ループ + 明示的な終了条件（break）を採用しました。  
- 数値の範囲を 1〜100 に限定することで、判定ロジックを単純化し、可読性を優先しています。  
- コメントでは「何をしているか」ではなく、「なぜその構造を採用したか」を意識して記述しています。
---
## 苦労した点
本作品は初めて作成した個人作品であるため、実装段階では試行錯誤しながら進める部分が多くありました。  
特に、第三者がコードを読んだ際にも設計意図や処理の流れが理解できるようにする点について、コード構造およびコメントの整理に時間を要しました。
